# 战斗自动脚本配置清单

## 必须配置的项目

### 1. 战斗区域配置（必需）
使用 `set_combat_regions()` 方法设置以下区域（坐标格式：x, y, w, h）：

```python
combat_script.set_combat_regions(
    enemy_region=(x, y, w, h),        # 敌军区域（左侧）
    ally_region=(x, y, w, h),         # 己方区域（右侧）
    main_char_region=(x, y, w, h),    # 主角区域
    general_region=(x, y, w, h),      # 武将区域
    turn_region=(x, y, w, h),         # 回合指示器区域（显示"25"的区域）
    right_button_region=(x, y, w, h), # 右侧按钮区域（技能/召唤/道具按钮）
    hp_bar_regions=[                  # 9个血量条检测区域（按顺序：主角1-3，武将1-6）
        (x1, y1, w1, h1),  # 主角1
        (x2, y2, w2, h2),  # 主角2
        (x3, y3, w3, h3),  # 主角3
        (x4, y4, w4, h4),  # 武将1
        (x5, y5, w5, h5),  # 武将2
        (x6, y6, w6, h6),  # 武将3
        (x7, y7, w7, h7),  # 武将4
        (x8, y8, w8, h8),  # 武将5
        (x9, y9, w9, h9),  # 武将6
    ]
)
```

### 2. 面板区域配置（推荐）
设置以下面板区域，提高识别准确性：

```python
combat_script.skill_panel_region = (x, y, w, h)   # 技能面板区域（点击技能按钮后弹出）
combat_script.item_panel_region = (x, y, w, h)    # 道具面板区域（点击道具按钮后弹出）
combat_script.summon_panel_region = (x, y, w, h)  # 召唤面板区域（点击召唤按钮后弹出）
```

### 3. 敌人固定点位配置（必需）
**重要：** 必须为每个账号设置3个敌人的固定点位坐标：

```python
# 为账号0设置敌人固定点位
combat_script.set_fixed_enemy_positions(
    account_index=0,
    enemy_positions=[
        (x1, y1),  # 敌人1的固定坐标
        (x2, y2),  # 敌人2的固定坐标
        (x3, y3),  # 敌人3的固定坐标
    ]
)

# 如果有多个账号，需要为每个账号设置
combat_script.set_fixed_enemy_positions(account_index=1, enemy_positions=[(x1, y1), (x2, y2), (x3, y3)])
combat_script.set_fixed_enemy_positions(account_index=2, enemy_positions=[(x1, y1), (x2, y2), (x3, y3)])
```

## 可选配置项

### 4. 控制开关（可选）
```python
combat_script.keep_support_general = True   # 是否保证辅助武将（刘备）在场，默认False
combat_script.enable_main_heal = True       # 主角是否使用恢复药加血，默认True
combat_script.enable_main_summon = True     # 主角是否召唤武将，默认True
```

### 5. 初始化战斗追踪（必需）
在战斗开始时调用：

```python
combat_script.init_combat_tracking()
```

## 配置示例代码

```python
# 创建战斗脚本实例
combat_script = CombatAutoScript(thread_instance)

# 1. 设置战斗区域
combat_script.set_combat_regions(
    enemy_region=(100, 100, 300, 400),      # 根据实际游戏界面调整
    ally_region=(500, 100, 300, 400),
    main_char_region=(520, 150, 200, 300),
    general_region=(520, 350, 200, 200),
    turn_region=(400, 50, 100, 30),
    right_button_region=(700, 100, 200, 500),
    hp_bar_regions=[
        (550, 160, 30, 15),  # 主角1血量条
        (580, 160, 30, 15),  # 主角2血量条
        (610, 160, 30, 15),  # 主角3血量条
        (550, 360, 30, 15),  # 武将1血量条
        (580, 360, 30, 15),  # 武将2血量条
        (610, 360, 30, 15),  # 武将3血量条
        (640, 360, 30, 15),  # 武将4血量条
        (670, 360, 30, 15),  # 武将5血量条
        (700, 360, 30, 15),  # 武将6血量条
    ]
)

# 2. 设置面板区域
combat_script.skill_panel_region = (750, 200, 150, 300)
combat_script.item_panel_region = (750, 200, 150, 300)
combat_script.summon_panel_region = (750, 200, 150, 300)

# 3. 设置敌人固定点位（必须配置）
combat_script.set_fixed_enemy_positions(
    account_index=0,
    enemy_positions=[
        (200, 200),  # 敌人1坐标（根据实际位置调整）
        (200, 300),  # 敌人2坐标
        (200, 400),  # 敌人3坐标
    ]
)

# 4. 初始化战斗追踪
combat_script.init_combat_tracking()

# 5. 启动战斗循环
combat_script.run_combat_loop()
```

- [ ] 已设置道具面板区域（item_panel_region，推荐）

## 注意事项

1. **坐标单位**：所有坐标都是像素坐标，相对于游戏窗口的左上角 (0, 0)
2. **区域格式**：所有区域都是 (x, y, w, h) 格式，x和y是左上角坐标，w和h是宽度和高度
3. **敌人点位**：敌人固定点位必须在战斗开始前设置，确保3个敌人的坐标准确
4. **血量条顺序**：9个血量条区域必须按照：主角1、主角2、主角3、武将1-6的顺序
5. **面板区域**：如果不设置面板区域，会使用 right_button_region 作为备用区域，但准确性可能降低

## 如何获取坐标

可以使用游戏窗口截图工具或大漠插件的截图功能，在游戏界面中测量各个区域和点的坐标。
