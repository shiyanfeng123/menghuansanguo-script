# 战斗逻辑修正说明

## 🔄 修正的核心逻辑

### 1. **召唤优先策略**
```
场上己方队伍中没有辅助武将 → 优先召唤辅助武将（刘备）
没有辅助武将 → 召唤输出武将（魔化关羽/曹操）
```

### 2. **技能CD检测机制**
- **技能找不到图标 = 技能在CD**
- 主角技能、药品、刘备清除状态技能都有CD
- 找不到技能图标则不使用，等待CD结束

### 3. **主角操作逻辑**
```
尝试使用技能1 → 在CD
尝试使用技能2 → 在CD
尝试使用技能3 → 在CD
全部CD → 使用药品给武将加血
```

### 4. **刘备技能释放策略**

#### 轮转顺序
1. 加攻击（对友军）
2. 加血（对友军）
3. 加速（对友军）
4. 控制（对敌军）
5. 清除状态（对敌军）

#### 优先级规则
```
检测到敌军有状态？
    ├─ 是 → 尝试清除状态
    │       ├─ 清除状态技能可用 → 使用清除
    │       └─ 清除状态技能CD → 等待（不执行任何操作）
    │
    └─ 否 → 按顺序使用辅助技能
            ├─ 技能可用 → 使用 + 切换到下一个
            └─ 技能CD → 保持当前位置（下次继续尝试）
```

## 📊 完整战斗流程

```
回合开始
    ↓
全局检测所有账号状态
    ↓
主角操作
    ├─ 尝试技能1
    ├─ 尝试技能2  
    ├─ 尝试技能3
    └─ 全部CD → 使用药品
    ↓
召唤判断
    ├─ 场上没有辅助武将 → 优先召唤刘备
    └─ 有辅助武将或刘备CD → 召唤输出武将
    ↓
武将操作
    ├─ 输出武将 → 群体攻击
    └─ 刘备
        ├─ 检测敌军状态
        ├─ 小肠状态 → 尝试清除
        └─ 无状态 → 按顺序释放辅助技能
```

## 🎯 关键改进点

### 1. **CD处理**
- 所有技能都能检测CD状态
- CD时保持待命，等待可用时再使用
- 主角技能全部CD时自动使用药品

### 2. **辅助技能循环**
- 按固定顺序释放：加攻击 → 加血 → 加速 → 控制 → 清除
- 成功使用后切换到下一个
- 当前技能CD时保持位置，下次继续

### 3. **清除状态优先级**
- 检测到敌军状态时优先清除
- 清除状态技能CD时等待，不执行其他操作
- 确保有状态时优先处理

## 💡 实际运行效果

### 场景1：正常战斗
```
回合1：刘备使用"加攻击"（成功）→ 下次用"加血"
回合2：刘备使用"加血"（成功）→ 下次用"加速"
回合3：刘备使用"加速"（成功）→ 下次用"控制"
回合4：刘备使用"控制"（成功）→ 下次用"清除状态"
回合5：刘备使用"清除状态"（成功）→ 下次用"加攻击"
...
```

### 场景2：检测到敌军状态
```
回合N：检测到敌军有状态
      → 尝试使用"清除状态"
      → 技能可用，成功清除
      → 保持当前位置，下次继续循环
```

### 场景3：清除状态技能CD
```
回合N：检测到敌军有状态
      → 尝试使用"清除状态"
      → 技能在CD（找不到图标）
      → 不执行任何操作，等待CD
回合N+1：清除状态技能可用
        → 成功使用清除状态
```

### 场景4：主角技能全部CD
```
回合N：主角尝试技能1 → CD
      → 尝试技能2 → CD
      → 尝试技能3 → CD
      → 全部CD
      → 自动使用药品给武将加血
```

## 🔧 代码实现要点

### `click_skill_icon` 方法改进
```python
def click_skill_icon(self, skill_name, dm_object, panel_region):
    # 找不到技能图标 → 返回False（技能在CD）
    if not pos or len(pos) == 0:
        return False  # 技能在CD
    return True  # 技能可用，成功使用
```

### `main_char_action` 方法改进
```python
# 遍历所有技能，找到第一个不在CD的技能使用
for skill_name in group_attack_skills:
    if self.use_skill_workflow(skill_name, dm_object, target_type='enemy'):
        skill_used = True
        break

# 全部CD则使用药品
if not skill_used:
    self.use_heal_item(dm_object)
```

### `general_action` 方法改进
```python
# 刘备技能释放逻辑
if has_status:
    # 有状态，尝试清除
    if not self.use_skill_workflow('清除状态', ...):
        # 清除技能在CD，等待
        pass
else:
    # 无状态，按顺序使用辅助技能
    skill_name = self.support_skill_sequence[self.current_skill_index]
    if self.use_skill_workflow(skill_name, ...):
        # 成功使用，切换到下一个
        self.current_skill_index += 1
    # CD则保持当前位置
```

## ✅ 优势

1. **智能CD管理** - 自动检测技能CD，避免无效操作
2. **循环优化** - 辅助技能循环释放，保持最佳效果
3. **优先级明确** - 清除状态优先于普通辅助技能
4. **完美闭环** - 主角技能CD时自动使用药品，形成循环
